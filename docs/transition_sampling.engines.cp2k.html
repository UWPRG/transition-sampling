Run the forward and reverse trajectories to get a shooting point.

Launch the MD simulation in both the forward and reverse direction from the assigned starting points and velocities, in parallel. These are spawned in new processes. Awaiting this waits for both simulations to commit to a basin or time out

class transition_sampling.engines.cp2k.CP2K_inputs.CP2KInputsHandler(cp2k_inputs_file)¶

Bases: object

Handles manipulating the raw CP2K Inputs data structure.

This class parses a CP2K inputs file into a memory, then supplies methods for altering it. The original file is not modified. Once the in-memory inputs have been changed, the current state can be written out into an equivalent CP2K inputs file.

Parameters

cp2k_inputs_file (str) – The cp2k inputs file that serves as a template for this class
